简要记录 Windows 与 Microsoft Office 的 KMS 激活方法。

<!--more-->

## Windows

管理员权限运行 CMD 或 Windows PowerShell，选择性输入如下命令：

### 查看 Windows 版本信息

`slmgr /dlv`

### 安装对应 GVLK

`slmgr /ipk <GVLK>`

### 设定 KMS 服务器

`slmgr /skms <KMS Server>`

### 激活 Windows

`slmgr /ato`

## Microsoft Office

以下操作，Microsoft Office 需为 VOL(批量授权)版。

首先找到 OSPP.vbs 文件所在文件夹，一般为 C:\Program Files\Microsoft Office\Office16。

按住 Shift，点击鼠标右键或按下 Application 键，选择 复制文件地址 菜单项目。

管理员权限打开 CMD 或 Windows PowerShell，利用 `cd` 命令切换到 OSPP.vbs 所在文件夹。

### 设定 KMS 服务器

`cscript ospp.vbs /sethst:<KMS Server>`

### 激活 Microsoft Office

`cscript ospp.vbs /act`

## 注意事项

1. Windows KMS 与 Microsoft Office KMS 必须保持一致，否则报错。

2. 确保 Windows 和 Microsoft Office 为批量授权版。

3. 各版本的 GVLK 官网可查，可自行检索。

4. 使用合规的 KMS 服务器，支持正版，从我做起(手动狗头)。
